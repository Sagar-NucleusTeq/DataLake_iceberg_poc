# Status Code of the Promotion
Delivery status code of the promotion material sent to the individual

| Values |     Def     |
|--------|-------------|

|   1    |  Sent       |
|   2    |  Failed     |
|   3    |  Taken into account by the service provider |
|   4    |  Received on mobile |
|   5    |  Pending    |
|   6    |   Delivery canceled |
|   7    |  Prepared   |
|   8    |  Sent to the service provider |
|   9    |  Unknown    |
| 127    |  Non applicable |
